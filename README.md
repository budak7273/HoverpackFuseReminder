Hover Packs draw 100 MW by default, but don't appear on the power chart, which can lead to confusing situations when trying to restart electric grids, especially small ones. Certain mods like [Archipelago](https://ficsit.app/mod/Archipelago) can give you a Hover Pack earlier in progression than expected, making this even more confusing. This mod adds a visible warning to the fuse reset UI if you're wearing a Hover Pack.

![Screenshot of the mod's warning](https://raw.githubusercontent.com/budak7273/HoverpackFuseReminder/refs/heads/main/ModpageResources/FuseboxScreenshot.jpg)

**Multiplayer Compatible!** Can work client side only.
